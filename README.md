# Generador de Cartas Aleatorias

Proyecto desarrollado como práctica de JavaScript, HTML y CSS.

---

## Descripción

Esta aplicación genera una carta aleatoria cada vez que se carga la página o cuando el usuario presiona el botón **Generar carta**.

Cada carta puede mostrar:

- Palos: ♥ ♦ ♠ ♣
- Valores: A, 2-10, J, Q, K

---

## Funcionalidades

- Carta aleatoria al iniciar la página.
- Botón para generar una nueva carta.
- Cambio automático de color según el palo:
  - Rojo: corazones y diamantes
  - Negro: picas y tréboles

---

## Cómo ejecutar el proyecto

1. Clona o descarga este repositorio:
   ```bash
   $ git clone https://github.com/ProyectBlackWolf3005/JoalexisPinto-Generador-de-Cartas-Aleatorias.git

2. Abre el archivo index.html en tu navegador o inicia un servidor local con VS Code (extensión Live Server).

3. Presiona el botón "Generar carta" para generar una nueva.
